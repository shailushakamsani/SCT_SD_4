### E-Commerce Product Information Scraper

This project extracts product information from an online e-commerce website and stores it in a CSV file.

### Features

- Extract Product Name
- Extract Product Price
- Extract Product Rating
- Save data into CSV format

- BeautifulSoup
- CSV
